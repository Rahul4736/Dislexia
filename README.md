 🧠 Dyslexia AI Assistant – Multilingual Accessibility System

📌 Project Overview

The Dyslexia AI Assistant is an AI-powered system designed to help individuals with dyslexia read and understand text more easily. It simplifies complex words, provides multilingual support, and converts text into speech.

 🎯 Features

* Upload PDF or Audio files
* AI-based text simplification
* Word splitting for readability
* Supports 22 Indian languages
* Audio output using Text-to-Speech

 🏗️ System Workflow

Input → Text Extraction → AI Processing → Output → Audio
